UnlMapAndroidSDK/com.mapbox.mapboxsdk.location/LocationComponentOptions/foregroundName

foregroundName

@Nullable
open fun foregroundName(): String

String image name, identical to one used in the first parameter of addImage, the component, will used this image in place of the provided or default mapbox_foregroundDrawable.

A maki-icon name (example: "circle-15") may also be provided. These are images that can be loaded with certain styles. Note, this will fail if the provided icon name is not provided by the loaded map style.

Return

String icon or maki-icon name
