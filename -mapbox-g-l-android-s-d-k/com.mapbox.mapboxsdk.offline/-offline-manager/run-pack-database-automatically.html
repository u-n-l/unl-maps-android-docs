runPackDatabaseAutomatically

open fun runPackDatabaseAutomatically(autopack: Boolean)

Sets whether database file packing occurs automatically. By default, the automatic database file packing is enabled.

If packing is enabled, database file packing occurs automatically after an offline region is deleted by calling delete or the ambient cache is cleared by calling clearAmbientCache. If packing is disabled, disk space will not be freed after resources are removed unless packDatabase is explicitly called.

Parameters

autopack

flag setting the automatic database file packing.
